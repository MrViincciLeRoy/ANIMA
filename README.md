# ANIMA
ANIMA is an anime-style AI companion with a web interface and speech synthesis. It uses the Groq API for text generation and Web Speech TTS for speech synthesis.
## Key Features
* Anime-style avatar with lip-sync and facial expressions
* Text-based conversation interface
* Speech synthesis using Web Speech TTS
* Emotion detection and response
## Tech Stack
* Frontend: HTML, CSS, JavaScript
* Backend: Groq API for text generation
* Speech Synthesis: Web Speech TTS
## Installation
1. Clone the repository: `git clone https://github.com/username/ANIMA.git`
2. Install dependencies: `npm install`
3. Start the application: `npm start`
## Usage
1. Open the application in a web browser: `http://localhost:3000`
2. Interact with the avatar using the text-based conversation interface
## Required Environment Variables
* `ANIMA_GROQ_KEY`: Groq API key for text generation
* `ANIMA_SPEECH_SYNTHESIS`: Enable or disable speech synthesis (default: enabled)
